## Technique template

Every technique entry requires:

1. Overview
2. Tactic + Protocols
3. Severity score (4 dimensions, composite)
4. Attack scenario (step by step)
5. Example request / payload
6. Real-world breach mapping
7. Detection (Sigma rule + behavioural indicators)
8. Remediation
9. References

Full format in [CONTRIBUTING.md](https://github.com/isha-singhMalik/Astra/blob/main/CONTRIBUTING.md#technique-file-format).
